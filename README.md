# Code_alpha

This repository contains two movie data analysis projects developed using Python and Jupyter Notebook.

## Project 1
Files:
- p1.ipynb
- mymoviedb.csv

Features:
- Data Cleaning
- Genre Analysis
- Popularity Analysis
- Data Visualization

## Project 2
Files:
- P2.ipynb
- netflix_titles.csv

Features:
- Netflix Dataset Analysis.
- Exploratory Data Analysis.
- Data preprocessing.
- Data Visualization.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook
